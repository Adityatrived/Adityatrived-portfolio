FIT LIFE

FitLife is a comprehensive gym website designed to provide a seamless experience for gym members and personal trainers. It includes multiple sections and functionalities to cater to different aspects of gym management and user interaction. The project utilizes HTML for the structure, CSS for styling, and JavaScript for dynamic functionalities.

Weather Dashboard

A weather dashboard that provides real-time weather information for any location. Users can search for a city to get current weather details, including temperature, humidity, and weather conditions, as well as a 5-day forecast. This project makes use of an external weather API, demonstrating how to handle API requests and responses using JavaScript, along with HTML for structure and CSS for styling.

Visit

MOVIE SEARCH

A movie search application that allows users to search for movies and view details such as the title, poster, release date, and rating. This project uses an external movie database API to fetch data and display it dynamically. HTML is used for the structure, CSS for styling, and JavaScript for handling API requests and displaying results.

Visit

Interactive Photo Gallery

An interactive photo gallery that lets users view images in a grid layout and click on them to see a larger version in a lightbox. This project demonstrates how to create a responsive and visually appealing gallery with HTML and CSS, and how to add interactivity with JavaScript to handle image clicks and lightbox functionality.
